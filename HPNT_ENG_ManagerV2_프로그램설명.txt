HPNT ENG Manager V2.0 프로그램 개요 및 주요 기능 정리

1. 프로그램 목적
- 건설/엔지니어링 프로젝트의 자재 요청 및 관리를 위한 경량화된 웹 기반 관리 시스템
- iPad, PC, 클라우드 등 다양한 환경에서 동작
- 실시간 DB 동기화(OneDrive 연동), 관리자 수동 동기화 지원

2. 주요 기술 스택
- Python 3, Flask 2
- SQLite3 (로컬/클라우드 DB)
- HTML5, CSS3, JavaScript(ES6)
- Render.com 클라우드 배포

3. 주요 기능
- 자재요청 등록/목록/상세/수정/삭제/복사/상태변경/검색/클론/인라인수정 등 CRUD
- 자재요청별 이미지(사진) 첨부 및 썸네일 표시
- 상태별 현황 대시보드, 키워드 검색, 테이블+카드 하이브리드 UI
- 전체 삭제 시 ID 재정렬(1번부터), DB 무결성 유지
- 관리자용 DB 수동 업로드/다운로드, 이미지 전체 zip 다운로드
- OneDrive 연동: db 및 images 폴더 실시간 동기화(로컬)
- 클라우드(Render) 환경: 서버 내 db/images 폴더 사용
- 브라우저/프록시 캐시 완전 무효화, PWA 서비스워커 비활성화
- 관리자 페이지에서 DB/이미지 백업 및 복원 가능

4. 주요 라우트(Flask)
- /               : 홈, 환경정보, DB위치 안내
- /requests       : 자재요청 목록(검색/필터/상태별 카운트)
- /add            : 자재요청 등록(이미지 첨부 지원)
- /request/<id>   : 상세조회/수정/삭제/복사/상태변경
- /admin/db-download : DB 파일 다운로드(관리자)
- /admin/db-upload   : DB 파일 업로드(관리자)
- /admin/images-download : 이미지 전체 zip 다운로드(관리자)
- /images/<파일명> : 개별 이미지 파일 다운로드/표시

5. DB/이미지 경로 정책
- 로컬: OneDrive/HPNT_Manager/db, OneDrive/HPNT_Manager/images
- 클라우드: 프로젝트/db, 프로젝트/db/images
